========================================
ИНСТРУКЦИЯ ПО ЗАГРУЗКЕ ИЗОБРАЖЕНИЙ ДЛЯ КУРСОВ
========================================

1. ПОДГОТОВКА ФАЙЛА
   -----------------
   Создайте Excel файл (images.xlsx) или CSV файл (images.csv) с двумя столбцами:
   
   Столбец 1: URL курса (например: https://skyeng.ru/yazykovye-kursy/ispanskij/)
   Столбец 2: URL скриншота (например: https://skrinshoter.ru/sYrSY478xVd)
   
   Пример Excel:
   A1: https://skyeng.ru/yazykovye-kursy/ispanskij/
   B1: https://skrinshoter.ru/sYrSY478xVd
   
   Пример CSV (разделитель - табуляция или запятая):
   https://skyeng.ru/yazykovye-kursy/ispanskij/,https://skrinshoter.ru/sYrSY478xVd

2. РАЗМЕЩЕНИЕ ФАЙЛА
   ----------------
   Поместите файл в одно из следующих мест:
   - C:\Users\Daniil\Downloads\images.xlsx (или images.csv)
   - Корень проекта (папка sravni.ru)\images.xlsx (или images.csv)

3. УСТАНОВКА ЗАВИСИМОСТЕЙ
   -----------------------
   Если библиотека xlsx еще не установлена, выполните:
   npm install

4. ЗАПУСК СКРИПТА
   --------------
   Выполните команду:
   npm run download-images
   
   Или напрямую:
   node scripts/download-course-images.js

5. ЧТО ДЕЛАЕТ СКРИПТ
   -----------------
   - Читает файл images.xlsx или images.csv
   - Для каждой строки:
     * Находит курс в базе данных по URL
     * Заходит на URL скриншотера
     * Извлекает прямую ссылку на изображение
     * Скачивает изображение
     * Сохраняет в папку public/uploads/courses/
     * Обновляет поле image у курса в базе данных

6. РЕЗУЛЬТАТ
   ---------
   Скрипт выведет статистику:
   - ✅ Успешно обновлено: количество курсов с обновленными изображениями
   - ⚠️  Не найдено курсов: количество URL, которые не найдены в базе
   - ❌ Ошибок: количество ошибок при скачивании
   - ⏭️  Пропущено: количество пустых или некорректных строк

7. ВАЖНО
   -----
   - Убедитесь, что все курсы уже загружены в базу данных
   - URL курсов должны точно совпадать с теми, что в базе данных
   - Скрипт автоматически обрабатывает редиректы и различные форматы URL скриншотеров
   - Изображения сохраняются с уникальными именами
