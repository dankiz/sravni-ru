📋 ПОШАГОВАЯ ИНСТРУКЦИЯ:

1️⃣  ОТКРОЙТЕ ТЕРМИНАЛ/КОМАНДНУЮ СТРОКУ в папке с сайтом
   
   Или перейдите в папку:
   cd "C:\Users\Daniil\Desktop\Скрипты\Skysmart\sravni.ru"

2️⃣  УСТАНОВИТЕ ЗАВИСИМОСТИ
   npm install
   
   ⏱️ Это займет несколько минут

3️⃣  УСТАНОВИТЕ И ЗАПУСТИТЕ PostgreSQL

   Если PostgreSQL не установлен:
   - Скачать: https://www.postgresql.org/download/windows/
   - Установите и запомните пароль для пользователя postgres
   
   Если PostgreSQL установлен, но не запущен:
   - Нажмите Win + R → введите: services.msc
   - Найдите "postgresql-x64-XX" (где XX - версия)
   - Правой кнопкой → "Запустить"
   
   📋 Подробнее: см. файл КАК_ЗАПУСТИТЬ_POSTGRESQL.txt

4️⃣  СОЗДАЙТЕ БАЗУ ДАННЫХ
   Откройте pgAdmin и создайте БД с именем: sravni_ru
   
   Или через командную строку:
   psql -U postgres -c "CREATE DATABASE sravni_ru;"

5️⃣  СОЗДАЙТЕ ФАЙЛ .env.local в корне проекта
   
   Создайте файл .env.local со следующим содержимым:
   
   DATABASE_URL="postgresql://postgres:ВАШ_ПАРОЛЬ@localhost:5432/sravni_ru?schema=public"
   NEXTAUTH_URL="http://localhost:3000"
   NEXTAUTH_SECRET="local-dev-key-12345"
   ADMIN_EMAIL="admin@example.com"
   ADMIN_PASSWORD="admin123"
   
   ⚠️ Замените ВАШ_ПАРОЛЬ на реальный пароль PostgreSQL

6️⃣  ИНИЦИАЛИЗИРУЙТЕ БАЗУ ДАННЫХ
   npx prisma generate
   npx prisma db push
   
7️⃣  СОЗДАЙТЕ АДМИНИСТРАТОРА
   
   Выполните ОДНУ из команд:
   
   npm run create-admin
   (или)
   npm run init-admin
   
   ⚠️ ВАЖНО: Без этой команды вы НЕ СМОЖЕТЕ войти в админ-панель!
   
   Команда создаст администратора:
   Email: admin@example.com
   Password: admin123

8️⃣  ЗАПУСТИТЕ САЙТ
   npm run dev

9️⃣  ОТКРОЙТЕ БРАУЗЕР
   http://localhost:3000
   
   Админ-панель: http://localhost:3000/admin
   Логин: admin@example.com
   Пароль: admin123
   
   ⚠️ Если не можете войти - выполните снова:
   npm run create-admin
   
   Или смотрите файл: СОЗДАТЬ_АДМИНА.txt

❓ ЧТО ТАКОЕ NEXTAUTH_SECRET?

Это секретный ключ для шифрования сессий.

Для ЛОКАЛЬНОЙ разработки можно использовать любой текст:
  NEXTAUTH_SECRET="мой-секретный-ключ-123"

Для генерации безопасного ключа:
  npm run generate-secret

Или онлайн: https://generate-secret.vercel.app/32
