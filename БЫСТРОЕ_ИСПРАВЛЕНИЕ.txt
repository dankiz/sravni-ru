🚀 ПОШАГОВОЕ ИСПРАВЛЕНИЕ:

1. ОТКРОЙТЕ ТЕРМИНАЛ в папке проекта
   
2. СОЗДАЙТЕ АДМИНИСТРАТОРА:
   npm run init-admin
   
   Должно вывести: "Admin created/updated: admin@example.com"

3. ПРОВЕРЬТЕ ФАЙЛ .env.local существует и содержит:
   
   ADMIN_EMAIL="admin@example.com"
   ADMIN_PASSWORD="admin123"
   
   (и другие переменные из КАК_ЗАПУСТИТЬ.txt)

4. ПЕРЕЗАПУСТИТЕ СЕРВЕР:
   - Остановите текущий (Ctrl+C)
   - Запустите снова: npm run dev

5. ПОПРОБУЙТЕ ВОЙТИ В АДМИНКУ:
   http://localhost:3000/admin
   Логин: admin@example.com
   Пароль: admin123
